# ToDo List
REST API example.

## Endpoints

|Method|Endpoint|Body params|
|-|-|-|
|GET|/todos||
|GET|/todos/:id||
|POST|/todos|id, task, done|
|PUT|/todos/:id|task, done|
|DELETE|/todos/:id||

## Tools
Flask / Python
